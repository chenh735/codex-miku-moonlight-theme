# Miku Theme Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the approved turquoise Miku music-monogram icon, apply it to the managed theme shortcuts, and leave a verified themed launcher ready on the Windows taskbar.

**Architecture:** A dependency-free PowerShell build script renders geometric PNG frames with `System.Drawing` and packages them into a multi-resolution ICO. The existing managed runtime copies the icon into `%LOCALAPPDATA%\CodexMikuMoonlightTheme\package-v1`, and the installer binds both theme shortcuts to that installed path. Static and binary-format tests protect the artifact and shortcut contract; live installation and Windows-shell inspection verify the user-facing result.

**Tech Stack:** Windows PowerShell 5.1, System.Drawing, WScript.Shell shortcuts, Node.js 22+ binary tests, existing PowerShell regression suite.

## Global Constraints

- Keep the official Codex shortcut and icon unchanged.
- Use the approved deep moonlight-blue/violet background, turquoise `M` plus music note, and one restrained white/cyan sparkle.
- Include 16, 20, 24, 32, 48, 64, 128, and 256 pixel ICO frames.
- Store the installed icon under the versioned managed package, never a temporary or workspace-only path.
- Do not add a launcher executable, registry mutation, service, scheduled task, or autostart entry.
- Use normal Windows shell pinning; if Windows blocks automation, require one explicit user right-click to pin.

Create `windows/tests/theme-icon.test.mjs` with a PNG reader and ICO directory parser:

```js
import fs from 'node:fs';
import path from 'node:path';
import { fileURLToPath } from 'node:url';

const root = path.dirname(path.dirname(fileURLToPath(import.meta.url)));
const pngPath = path.join(root, 'assets', 'miku-music-mark.png');
const icoPath = path.join(root, 'assets', 'miku-music-mark.ico');
const required = [16, 20, 24, 32, 48, 64, 128, 256];

function assert(condition, message) {
  if (!condition) throw new Error(message);
}

const png = fs.readFileSync(pngPath);
assert(png.subarray(0, 8).equals(Buffer.from([137, 80, 78, 71, 13, 10, 26, 10])), 'Invalid PNG signature');
assert(png.readUInt32BE(16) === 1024 && png.readUInt32BE(20) === 1024, 'Reference PNG must be 1024x1024');

const ico = fs.readFileSync(icoPath);
assert(ico.readUInt16LE(0) === 0 && ico.readUInt16LE(2) === 1, 'Invalid ICO header');
const count = ico.readUInt16LE(4);
assert(count === required.length, `Expected ${required.length} ICO frames, got ${count}`);
const sizes = [];
for (let index = 0; index < count; index += 1) {
  const entry = 6 + index * 16;
  const width = ico[entry] || 256;
  const height = ico[entry + 1] || 256;
  const length = ico.readUInt32LE(entry + 8);
  const offset = ico.readUInt32LE(entry + 12);
  assert(width === height, `ICO frame ${index} is not square`);
  assert(offset + length <= ico.length, `ICO frame ${index} exceeds file bounds`);
  assert(ico.subarray(offset, offset + 8).equals(Buffer.from([137, 80, 78, 71, 13, 10, 26, 10])), `ICO frame ${index} is not PNG encoded`);
  sizes.push(width);
}
assert(JSON.stringify(sizes) === JSON.stringify(required), `Unexpected ICO sizes: ${sizes.join(', ')}`);
console.log('PASS: Miku music-mark PNG and ICO dimensions are valid.');
```

- [ ] **Step 3: Implement the dependency-free renderer**

Create `windows/scripts/build-theme-icon.ps1`. It must load `System.Drawing`, render each requested size using only geometric paths, encode each frame as PNG, and write an ICO directory whose offsets point to the concatenated PNG payloads. The core public contract is:

```powershell
[CmdletBinding()]
param([string]$OutputDirectory = (Join-Path (Split-Path -Parent $PSScriptRoot) 'assets'))

$ErrorActionPreference = 'Stop'
Add-Type -AssemblyName System.Drawing

$sizes = @(16, 20, 24, 32, 48, 64, 128, 256)
$pngPath = Join-Path $OutputDirectory 'miku-music-mark.png'
$icoPath = Join-Path $OutputDirectory 'miku-music-mark.ico'

function New-RoundedRectanglePath {
  param([System.Drawing.RectangleF]$Bounds, [float]$Radius)
  $path = [System.Drawing.Drawing2D.GraphicsPath]::new()
  $diameter = $Radius * 2
  $path.AddArc($Bounds.Left, $Bounds.Top, $diameter, $diameter, 180, 90)
  $path.AddArc($Bounds.Right - $diameter, $Bounds.Top, $diameter, $diameter, 270, 90)
  $path.AddArc($Bounds.Right - $diameter, $Bounds.Bottom - $diameter, $diameter, $diameter, 0, 90)
  $path.AddArc($Bounds.Left, $Bounds.Bottom - $diameter, $diameter, $diameter, 90, 90)
  $path.CloseFigure()
  return $path
}

function New-MikuIconPng {
  param([Parameter(Mandatory = $true)][int]$Size)
  $bitmap = [System.Drawing.Bitmap]::new($Size, $Size, [System.Drawing.Imaging.PixelFormat]::Format32bppArgb)
  $graphics = [System.Drawing.Graphics]::FromImage($bitmap)
  try {
    $graphics.SmoothingMode = [System.Drawing.Drawing2D.SmoothingMode]::AntiAlias
    $graphics.CompositingQuality = [System.Drawing.Drawing2D.CompositingQuality]::HighQuality
    $bounds = [System.Drawing.RectangleF]::new(0, 0, $Size, $Size)
    $backgroundPath = New-RoundedRectanglePath -Bounds $bounds -Radius ($Size * 0.23)
    $background = [System.Drawing.Drawing2D.LinearGradientBrush]::new(
      $bounds,
      [System.Drawing.Color]::FromArgb(255, 13, 28, 76),
      [System.Drawing.Color]::FromArgb(255, 91, 55, 157),
      45
    )
    try { $graphics.FillPath($background, $backgroundPath) } finally { $background.Dispose(); $backgroundPath.Dispose() }

    $mPath = [System.Drawing.Drawing2D.GraphicsPath]::new()
    $mPath.AddLines([System.Drawing.PointF[]]@(
      [System.Drawing.PointF]::new($Size * 0.23, $Size * 0.72),
      [System.Drawing.PointF]::new($Size * 0.31, $Size * 0.30),
      [System.Drawing.PointF]::new($Size * 0.50, $Size * 0.55),
      [System.Drawing.PointF]::new($Size * 0.67, $Size * 0.30),
      [System.Drawing.PointF]::new($Size * 0.75, $Size * 0.72)
    ))
    $mPen = [System.Drawing.Pen]::new([System.Drawing.Color]::FromArgb(255, 91, 235, 224), [Math]::Max(1.5, $Size * 0.105))
    $mPen.StartCap = $mPen.EndCap = [System.Drawing.Drawing2D.LineCap]::Round
    $mPen.LineJoin = [System.Drawing.Drawing2D.LineJoin]::Round
    try { $graphics.DrawPath($mPen, $mPath) } finally { $mPen.Dispose(); $mPath.Dispose() }

    $notePen = [System.Drawing.Pen]::new([System.Drawing.Color]::White, [Math]::Max(1, $Size * 0.045))
    $noteBrush = [System.Drawing.SolidBrush]::new([System.Drawing.Color]::White)
    try {
      $graphics.DrawLine($notePen, $Size * 0.70, $Size * 0.31, $Size * 0.70, $Size * 0.58)
      $graphics.DrawLine($notePen, $Size * 0.70, $Size * 0.31, $Size * 0.82, $Size * 0.27)
      $graphics.FillEllipse($noteBrush, $Size * 0.59, $Size * 0.54, $Size * 0.15, $Size * 0.11)
      $spark = [System.Drawing.Pen]::new([System.Drawing.Color]::FromArgb(235, 220, 255, 255), [Math]::Max(1, $Size * 0.018))
      try {
        $graphics.DrawLine($spark, $Size * 0.80, $Size * 0.13, $Size * 0.80, $Size * 0.24)
        $graphics.DrawLine($spark, $Size * 0.745, $Size * 0.185, $Size * 0.855, $Size * 0.185)
      } finally { $spark.Dispose() }
    } finally { $notePen.Dispose(); $noteBrush.Dispose() }

    $memory = [System.IO.MemoryStream]::new()
    $bitmap.Save($memory, [System.Drawing.Imaging.ImageFormat]::Png)
    return $memory.ToArray()
  } finally {
    $graphics.Dispose()
    $bitmap.Dispose()
  }
}

[System.IO.Directory]::CreateDirectory([System.IO.Path]::GetFullPath($OutputDirectory)) | Out-Null
[System.IO.File]::WriteAllBytes($pngPath, (New-MikuIconPng -Size 1024))
$frames = @($sizes | ForEach-Object { [pscustomobject]@{ Size = $_; Bytes = New-MikuIconPng -Size $_ } })
$stream = [System.IO.File]::Open($icoPath, [System.IO.FileMode]::Create, [System.IO.FileAccess]::Write)
$writer = [System.IO.BinaryWriter]::new($stream)
try {
  $writer.Write([uint16]0); $writer.Write([uint16]1); $writer.Write([uint16]$frames.Count)
  $offset = 6 + (16 * $frames.Count)
  foreach ($frame in $frames) {
    $writer.Write([byte](if ($frame.Size -eq 256) { 0 } else { $frame.Size }))
    $writer.Write([byte](if ($frame.Size -eq 256) { 0 } else { $frame.Size }))
    $writer.Write([byte]0); $writer.Write([byte]0)
    $writer.Write([uint16]1); $writer.Write([uint16]32)
    $writer.Write([uint32]$frame.Bytes.Length); $writer.Write([uint32]$offset)
    $offset += $frame.Bytes.Length
  }
  foreach ($frame in $frames) { $writer.Write([byte[]]$frame.Bytes) }
} finally { $writer.Dispose(); $stream.Dispose() }

Write-Host "Generated $pngPath and $icoPath"
```
